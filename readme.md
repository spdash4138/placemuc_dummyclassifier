# PlaceMux Phase 1 - Task 1

## Objective
Set up a reproducible machine learning environment and validate an end-to-end ML pipeline.

## Dataset
FIFA World Cup 2026 Player Performance Dataset

The dataset contains player attributes, match statistics, goals, assists, passing accuracy, defensive actions, disciplinary records, physical metrics, and advanced analytical indicators.

## Workflow
1. Data ingestion
2. Dataset validation
3. Train/Validation/Test split
4. DummyClassifier baseline model
5. Accuracy evaluation
6. Metrics logging
